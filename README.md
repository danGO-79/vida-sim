## 🧬 **Tu planta es irrepetible**

Cuando nace una nueva planta, su **ADN se genera al azar**. Diez genes determinan cómo crece, cuánto vive, cuántas flores produce y cómo resiste al clima.

Según qué rasgos dominen, desarrolla una **personalidad** propia:

```
🌿 Vertical     — crece alta y esbelta, desafiando la gravedad
🌿 Frondosa     — se llena de hojas y ramas, testigo de su vigor
🌿 Florecedora  — cubierta de flores cuando llega su momento
🌿 Resistente   — sobrevive condiciones adversas sin quebrarse
🌿 Productiva   — frutos abundantes, generosa con su entorno
🌿 Longeva      — lenta pero imparable, acumula años
🌿 Ahorradora   — usa cada gota con precisión
🌿 Enana        — compacta pero intensa, llena de vida
🌿 Equilibrada  — versátil en todo, especialista en nada
```

Cada planta tiene además su propia **paleta de color**: tallo, hojas, flores y frutos combinan en tonos que no se repiten. Ocho paletas base se entrelazan para crear miles de combinaciones posibles.

> No hay dos plantas iguales. Nunca. Cada una es una aventura.

## 🌱 **Crece, respira y envejece**

La planta no es decoración: **vive en un ciclo continuo** que responde a su entorno.

- **Arquitectura ramificada** — el crecimiento no es un contador abstracto: tallo, ramas laterales y ramillas emergen nodo a nodo. Cuando una rama alcanza su longitud, se vuelve **madura** y deja de alargarse; las flores y los frutos aparecen en los nodos concretos donde la planta los sostiene.
- **Biomasa y metabolismo** — crece cuando tiene energía, agua y nutrientes del suelo; se ralentiza al acercarse a su tamaño máximo, como en la naturaleza. La lluvia acelera la absorción de nutrientes; las hormigas del suelo también aportan un pequeño bonus.
- **Hojas progresivas** — aparecen poco a poco a lo largo de cada rama; una planta joven tiene pocas hojas separadas, una madura llena el aire de verde.
- **Otoño e invierno** — en otoño caen solo las **hojas**, nunca las ramas. Las hojas muertas se convierten en partículas y en acolchado que devuelve nutrientes al suelo. En invierno el follaje sigue reduciéndose con calma.
- **Primavera** — las ramas existentes **repueblan sus nodos** con brotes nuevos; no hace falta volver a crecer desde cero.
- **Estado latente** — si salud, hidratación y nutrientes caen demasiado bajo, la planta entra en reposo profundo. No muere: recupera agua y nutrientes poco a poco y despierta cuando las condiciones mejoran. No hay game over.
- **Estaciones con carácter** — la primavera acelera el crecimiento y trae flores; el verano madura los frutos; el otoño tiñe de dorado; el invierno invita al reposo.

## 🌦 **Un clima con memoria**

El tiempo no es aleatorio. En Vida, el clima es un **organismo vivo**: cada variable habla con las demás, y lo que ves ahora suele anunciar lo que viene después. No hay tiradas de dados — hay **coherencia atmosférica**.

---

### La brújula del tiempo: presión atmosférica

La presión no salta de golpe. **Sube y baja con suavidad**, como en la vida real, respondiendo a:

- **La estación** — cada época del año tiene su carácter propio
- **La hora del día** — mínima al amanecer, más alta a media mañana
- **La temperatura** — el aire frío pesa más; el calor lo alivia
- **La humedad** — una masa de aire húmedo tiende a presionar menos
- **Las nubes** — un cielo cubierto empuja la presión a la baja; uno despejado favorece la estabilidad
- **El viento** — corrientes secas elevan la presión; las húmedas la reducen
- **Las tendencias** — si la presión sube, tiende a seguir subiendo; si baja, el ciclo continúa

Cuando una borrasca se prolonga demasiado, la atmósfera **recupera presión por sí sola** y abre paso a días más serenos. El clima recuerda lo que ha pasado.

---

### El viento como consecuencia física

El viento no aparece porque sí. Nace cuando existe una **diferencia de presión** entre dos zonas: el aire siempre se mueve desde donde hay más hacia donde hay menos. Cuanto mayor la diferencia, más intenso el viento.

En Vida, el viento surge de causas reales:

| Causa | Qué produce |
|-------|-------------|
| **Presión estable** (<2 hPa en 24h) | Calma — apenas hay movimiento |
| **Presión descendiendo** (2–5 hPa) | Viento que crece gradualmente |
| **Descenso rápido** (>5 hPa) | Temporal posible — borrasca acercándose |
| **Borrasca profunda** (<995 hPa) | Vientos fuertes, ambiente inestable |
| **Contraste térmico diario** | Mañana suave → Mediodía activo → Tarde intenso → Noche quieta |
| **Estación** | Invierno y otoño más turbulentos; verano estable con brisas térmicas |
| **Nubosidad creciente** | El viento se refuerza mientras las nubes se forman |
| **Tormenta activa** | Lluvia intensa + presión bajando + humedad alta → rachas de 8–20 m/s |

El sistema rastrea la **evolución de la presión durante 48 horas** (dos días completos) para determinar si la atmósfera está tranquila, en cambio gradual o en espiral de tormenta. Las borrascas no duran para siempre: cuando se agotan, la presión se recupera y el viento amaina.

El panel lateral muestra la velocidad en **km/h** para que puedas leer el estado del aire con la misma facilidad que la temperatura.

---

### Sistema predictivo natural de lluvia

La lluvia **nunca aparece de la nada**. Existe una secuencia atmosférica que puedes leer con la mirada:

```
1. Sequedad      → Cielo estable, nada indica precipitación
2. Aviso         → La humedad sube, la presión baja, las nubes se forman
3. Lluvia        → La precipitación llega, moderada pero presente
4. Tormenta      → Presión baja, humedad alta, el cielo se enciende
5. Recuperación  → La borrasca se agota, el sol vuelve, el ciclo se reinicia
```

El panel lateral muestra la **probabilidad de lluvia** en tiempo real. No es decorado: el sistema evalúa muchos factores a la vez para calcularla:

| Señal | Qué significa |
|-------|---------------|
| Estación y mes | Primavera y otoño son húmedos; el verano, más seco |
| Humedad relativa | Cuanto más húmedo el aire, más probable la lluvia |
| Nubosidad | Un cielo cerrado aumenta las posibilidades; uno despejado las reduce |
| Presión atmosférica | Presión alta frena la lluvia; presión baja la invita |
| Viento | Brisas moderadas pueden anunciar borrasca; vientos extremos la dispersan |
| Tendencias | Humedad y nubes subiendo + presión bajando = aviso claro |
| Calor estival | Tarde calurosa y húmeda en verano puede desencadenar tormenta convectiva |

Si observas con calma, **aprendes a leer el cielo** antes de que caiga la primera gota.

---

### Borrascas que nacen y mueren

Las tormentas no duran eternamente. Cuando la lluvia se prolonga, el sistema entra en **fase de agotamiento**: la precipitación se debilita, la presión se recupera y la humedad post-lluvia se evapora poco a poco. Así el ciclo puede reiniciarse y el sol volver a brillar.

Tras días secos, el cielo también necesita **recuperarse** antes de volver a llover — como si la atmósfera guardara memoria de la última borrasca.

**Garantía de lluvia tras sequía prolongada:** el sistema cuenta los días consecutivos sin precipitación. A partir del día 18 la probabilidad sube gradualmente; al llegar a **35 días secos**, la lluvia se vuelve inevitable. Ninguna planta queda atrapada para siempre bajo un cielo cerrado.

---

### Relámpagos como presagio

Los relámpagos no acompañan cualquier llovizna. Aparecen cuando la tormenta **se está gestando o ya desatada**:

- Con lluvia moderada y alta probabilidad → destellos de **aviso**, antes del estallido
- Con lluvia intensa → tormenta activa, relámpagos en pleno apogeo
- Con probabilidad muy alta → la tormenta casi segura, el cielo se ilumina con más fuerza

El **sol responde al clima**: se desvanece cuando empieza a llover y desaparece con la lluvia moderada. Cuando la tormenta cesa, regresa con una transición suave — como si el día respirara de nuevo.

---

### Nubes con vida propia

Las nubes no son fondo estático. Tienen **ciclo de vida**:

- **Despejado** — pocas nubes, movimiento pausado, pausas frecuentes
- **Nublado** — más nubes, más oscuras, movimiento continuo
- **Tormenta** — el cielo se llena de nubes densas

Cada nube tiene velocidad propia, puede detenerse un momento y continuar, o desvanecerse mientras otras aparecen. El sistema no fuerza el clima: **las nubes responden a él**.

---

### Estaciones que lo cambian todo

El clima no es el mismo en primavera que en invierno. Cada estación modifica luz, temperatura y probabilidad de lluvia:

```
🌸 Primavera  — Crecimiento acelerado, flores, vitalidad, lluvias frecuentes
☀️ Verano     — Frutos que maduran, tardes calurosas, tormentas convectivas
🍂 Otoño      — Colores dorados, hojas que caen, atmósfera húmeda
❄️ Invierno   — Reposo, conservación, días cortos, nieve posible
```

La temperatura sigue el ritmo del sol: sube de día, baja de noche, y las nubes pueden enfriar el ambiente. El viento responde a la presión, la estación y el contraste térmico — todo encaja.


## 🌈 **El cielo está vivo**

El cielo cambia de color según la hora y la estación. Siete momentos del día se mezclan sin cortes:

```
🌌 Noche       — Azul profundo, casi negro, salpicado de estrellas
🌅 Amanecer    — Naranjas y rosas suaves; luz tenue, sin el brillo cegador del mediodía
🌤 Mañana      — Azul claro, fresco, lleno de promesas
☀️ Mediodía    — Azul intenso, el sol en su cenit
🌆 Tarde       — Tonos cálidos que anuncian el ocaso
🌇 Atardecer   — Rojos, naranjas, fuego en el horizonte
🌃 Crepúsculo  — Azul añil, la noche se acerca
```

El **sol** traza un arco estacional: alto en verano, bajo en invierno. Su color cambia con la elevación — de blanco en el cenit a naranja intenso en el horizonte. Al **amanecer** la paleta es distinta y el halo solar más contenido que a mediodía; la nubosidad atenúa la luz de forma coherente en cada fase del día.

Las **estrellas** aparecen gradualmente al caer la noche y se esconden tras las nubes cuando llega la tormenta.

## 🐞 **Visitantes del ecosistema**

Seis criaturas aparecen según la hora, la estación y el estado de la planta. Cada una con su propio ritmo:

```
🐝 Abeja      — Vuela de día; zigzaguea entre flores y frutos
🦋 Mariposa   — Aleteo grácil en las horas claras
🐞 Mariquita  — Vuela hacia la maceta, aterriza un momento y vuelve al cielo
🪲 Luciérnaga — Brilla en la oscuridad de la noche
🦟 Mosquito   — Merodea al anochecer en primavera
🐜 Hormiga    — Cruza el suelo en pequeños grupos, indiferente al espectáculo
```

Cuando **llueve, nieva o hay viento extremo**, los visitantes se refugian: el escenario queda en silencio.

## 🖥 **Interfaz invisible**

Un panel lateral discreto muestra el estado sin interrumpir la contemplación:

- 🌤 **Clima** — Estación, hora, sol, temperatura, humedad, viento, lluvia y **probabilidad de precipitación**
- 🌿 **Planta** — Salud, vitalidad, energía, hidratación, nutrientes, edad, hojas, flores, frutos y personalidad

Un botón ◀ colapsa el panel y la ventana se reduce a **400×400**: solo la planta, sin distracciones.

Cada pocos segundos, una **voz narrativa** describe el momento: *"Las hojas se mecen con la brisa de verano"*, *"La nieve cubre suavemente la maceta"*, *"Un grupo de hormigas cruza el suelo"*, *"La planta descansa en silencio"*.

Un **historial** registra los hitos: primer brote, primera flor, momentos de reposo y recuperación.

## 💾 **El recuerdo persiste**

La planta **guarda su estado automáticamente**. Al volver, continúa exactamente donde la dejaste: su edad, su salud, su arquitectura ramificada, su clima y su historia.

En macOS la partida se guarda en:

```
~/Library/Application Support/Vida/vida-save.json
```

Antes de cada guardado se crea una copia de seguridad en `vida-save.json.bak`.

El tiempo siguió pasando mientras no mirabas.

## 🛠 **Desarrollo y distribución**

Monorepo con npm workspaces. Requisitos: Node.js 18+.

```bash
npm install          # instalar dependencias
npm run dev          # ventana de desarrollo (Electron + hot reload)
npm run test         # tests unitarios
npm run typecheck    # comprobación de tipos
npm run package:mac  # build + .dmg para macOS (arm64)
npm run package:win  # instalador NSIS para Windows
npm run package:linux # AppImage y .deb para Linux
```

Los instaladores se generan en `apps/desktop/dist/` (por ejemplo `Vida-0.6.0-arm64.dmg`). Ejecuta siempre `npm run package:*` desde la raíz del repo; no empaquetes con `electron-builder` directamente en la carpeta raíz del monorepo.
